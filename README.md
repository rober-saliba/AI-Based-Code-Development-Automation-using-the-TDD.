# AI-Based-Code-Development-Automation-using-the-TDD.

This project introduces a fully automated software development system that integrates AI using Test-Driven Development (TDD) approach. The system receives specifications and test cases-specifically utilizing unit and functional tests to create an active feedback loop. Rather than repeatedly regenerating entire source files, it applies incremental code updates to safely modify only the required parts of the codebase. It autonomously executes the AI-generated code, captures compilation or logical errors, and feeds them back to the AI for iterative correction until all of the tests passes.
